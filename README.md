# 2D-RoboticsArm
A 2D robot arm simulator built from scratch using Python.
Click anywhere on the screen and the arm moves there automatically
using inverse kinematics!

## What This Project Does
- Click any point on screen
- Arm automatically calculates joint angles
- Moves to that exact position instantly

## Concepts Implemented
- Forward Kinematics — calculates hand position from joint angles
- Inverse Kinematics — calculates joint angles from target position
- Geometric approach using cosine rule and arctan2
- Real time visualization with mouse click control

## Technologies Used
- Python
- NumPy
- Matplotlib

## How To Run
pip install numpy matplotlib
python inversekinmatics.py

## What I Learned
Built this project to understand core robotics concepts
including forward and inverse kinematics from scratch
